# aliyun-cdn-refresh

## Usage

### From command line

```shell
npm install aliyun-cdn-refresh -g

aliyun-cdn-refresh --ObjectPath * --ObjectType * --AccessKeyId * --AccessKeySecret *
```

### From code

```js
// npm install aliyun-cdn-refresh

const operate = require('aliyun-cdn-refresh');

operate({
  ObjectPath: *,
  ObjectType: *,
  AccessKeyId: *,
  AccessKeySecret: *,
});
```
